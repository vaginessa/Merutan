Top 5 Best External Hard Drives in India

Did your computer or laptop run out of storage and you need to store your files, music, videos etc? What will you do now?

Well, this is the best time to buy extra storage. For as low as ₹4000, you can add on 1TB (terabyte) of storage to your computer or laptop using an external drive. 1 TB is more than enough to hoard lakhs of audio files or hundreds of movie files. Depending on your requirements, there are lots of external hard drives available in the market. But which one to choose?

We have researched multiple external hard drives and compiled the following list to help you find the best external drive corresponding to your requirements. So, Keep Calm and Read On.

1 . Seagate Backup Plus Slim

Seagate Backup Plus Slim is the best external drive for most users because it offers an ideal balance of speed, performance and price. It is affordable, slimmer and lighter than most of its competitors. It supports both Mac and Windows and includes Seagate Dashboard Backup Utility that allows the user to automatically backup their computer when the drive is connected. It comes with a free 200GB OneDrive cloud storage subscription valid for 2 years.

It weighs just 159g and is less than 0.5 inches thick. It is available in black, blue, red. The drive offers a lot of capacity (1 TB, 2 TB and 4 TB versions are available). It allows quick file transfer with USB 3.0 connectivity and allows USB 2.0 connectivity as well. It provides USB 3.0 read and write speeds of upto 100 MBps.

Specifications :

Capacity : 1TB / 2TB / 4TB
Connectivity : USB 3.0 (backwards compatible with USB 2.0)
Dimensions : 11.4 x 7.6 x 1.2 cm
Weight : 159g
Warranty : 3 years

2 . WD My Passport (2016)

WD My Passport is an affordable drive that offers outstanding performance. It supports both Windows and Mac but you need to reformat it to use it with a Mac. It includes WD’s SmartWare software that allows the user to automatically backup their computer when the drive is connected. You can easily recover deleted files. It also offers password protection and 256-bit data encryption.

It weighs just 245g and is 1.6cm thick. It is available in black, blue, red, orange, white and yellow. The drive offers a lot of capacity (1 TB, 2 TB, 3TB and 4 TB versions are available). It allows quick file transfer with USB 3.0 connectivity and allows USB 2.0 connectivity as well. It provides USB 3.0 read and write speeds of upto 100 MBps.

3 . WD Elements

WD Elements is a compact hard drive that won’t dig a hole in your pocket. It supports both Windows and Mac but you need to reformat it to use it with a Mac. It includes a trial version of WD’s SmartWare software that allows the user to automatically backup their computer when the drive is connected. It doesn’t come with software or hardware encryption.

It weighs 272g and is 1.5cm thick. It is available in black. The drive is available in 1 TB and 2 TB variants. It allows file transfer with USB 3.0 connectivity and allows USB 2.0 connectivity as well. It provides low USB 3.0 read and write speeds. There are different variants available online all of which provide the same performance, the only difference being the year of manufacture and available capacity.
